Overview

A secure console application developed entirely in MIPS Assembly language using the MARS Simulator. This project demonstrates an understanding of low-level hardware logic, memory management, and conditional branching.
Key Features

    Custom Logic: Implemented an odd-code filtering display function directly at the assembly level.

    User Input Validation: Secure handling of user inputs via the Run I/O console.

Demonstration

    Download the MARS MIPS Simulator.

    Open vault_console.asm in the simulator.

    Click Run > Assemble (F3).

    Click Run > Go (F5) and interact via the Run I/O terminal.
